2D FIGHTING GAME

Built with JavaScript, HTML5 Canvas, and CSS

Game Project

PROJECT OVERVIEW

This project is a 2D fighting game where players control characters with smooth animations, responsive controls, and dynamic combat mechanics. The game is developed using JavaScript, HTML5 Canvas for rendering, and animation effects.

Players can engage in combat with real-time health bar updates, a countdown timer, and precise collision detection to determine the outcome of each fight. The project showcases advanced front-end development techniques and game logic implementation.


KEY FEATURES:

  • Dynamic sprite animations for various character actions, including running, jumping, attacking, and taking hits.
  • Real-time health bar updates reflecting damage taken during combat.
  • Countdown timer to add urgency to each battle.
  • Responsive controls for both players, enabling smooth and intuitive gameplay.
  • Collision detection to determine whether attacks land successfully or miss.
