Mantid Project

XML Research on the Neutron and Muon Source project (work experience)

At the Rutherford Appleton Laboratory, I was introduced to the Mantid
Project. This project focused on providing the resources for scientists
to use high-performance computing and visualisation of materials within
operating the particle accelerator.

This work experience is one of the UK's seven publicity funded by the government.
This experience offered me a uniqueness of the diversity of world-leading expertise,
which drives my knowledge and inspiration towards science & technology forward which
allowed maximising my impact towards understanding this area.

I was shown how the scientist uses these tools and how they are developed. I had the amazing
opportunity to participate with a group of developers working on the documentation area of the
project which helped the scientists and PhD students understand how to use the tools correctly.
The project I was working on for 2 weeks was to document unique sources of both pulsed neutrons and muons
for exploring properties of matter by measuring the locations of atoms and also documenting how to use the tools
the mantid project offers where I wrote a guide how to create XML files. This contribution has allowed me to
make a difference for the first time.

What I've Leanrt

  • I learnt the importance of why documentation is important when working on any project, as it allows
    clear guidance and understanding for the users the product/service is intended to be used by.
  • How to write a step by step guide and really breakdown the entire process for anyone to be able to
    follow these instructions easily.
  • Different practices and skills that are necessary for a group when working on a project. Such as methods
    like the waterfall and agile principles. Skills such as time management and organisational skills.
